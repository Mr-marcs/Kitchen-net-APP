# Kitchen-net

Um APP de cozinhar para o TCC usando React Native e Node.

![Logo](https://pipe.miroware.io/60a47afdab37f801c2e52bb1/TCC/logo.png)

## Como iniciar:

Colocar o seguinte comando no CMD e vai rodar o client do expo:

``` bash
$ npm start
```
